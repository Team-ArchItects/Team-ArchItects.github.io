Maintainable

We have placed all of the data that needs to be changed in order to maintain the site inside a single data.js file. This allows for the site to be maintained and updated by a layperson.

Future-proof

React/Next.js/Tailwind are all modern technologies which will add to the longevity of the site.

Scalable

We have built the site to be scalable by adding as much as possible into React components. This ensures reusable code and the scalability of the website.

Minimalist

We have taken care to ensure the website and documentation are minimalist and clean, to ensure that they are clear for users.

Modern

The website is modern and implements modern SAAS soltuions and APIs.

Characterful

The website is designed with an industrial aesthetic, inspired by Birmingham.
